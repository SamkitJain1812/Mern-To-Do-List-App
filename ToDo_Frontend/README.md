**# 📝 ToDo Frontend**

A simple and clean Task Management frontend built with React and Tailwind CSS. This application allows users to create, view, filter, and manage their tasks with ease. It interacts with a Node.js + MongoDB backend via REST APIs.

---

## ✨ Features

- Create, update, and delete tasks
- Filter tasks by priority
- User authentication with JWT
- Responsive UI with Tailwind CSS

---

## ⚙️ Technologies Used

- **React** – Frontend framework
- **Tailwind CSS** – Styling
- **Fetch API** – Making HTTP requests
- **Render** – Deployment platform
