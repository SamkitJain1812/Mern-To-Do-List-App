# 🧠 ToDo Backend API

This is the backend API for the **ToDo Application**, built with **Node.js**, **Express**, and **MongoDB (Mongoose)**. It provides RESTful endpoints for user authentication and task management.

---

## 📦 Features

- ✅ User registration and login (with JWT-based authentication)
- 📝 Create, Read, Update, and Delete (CRUD) operations for tasks
- 🔐 Protected routes using middleware
- 🌐 MongoDB Atlas for cloud database

---

## 🚀 Live API

🌐 [https://todo-backend-2pp1.onrender.com](https://todo-backend-2pp1.onrender.com)

---

## 🔧 Technologies Used

- **Node.js** – Runtime
- **Express.js** – Web framework
- **MongoDB + Mongoose** – NoSQL Database
- **JWT** – Authentication
- **bcryptjs** – Password hashing
- **dotenv** – Environment configuration
- **Render** – Deployment platform
